# automatic-lyrics
A program that displays the song lyrics automatically on terminal as the song plays in Spotify. This also continues when it moves to another song.

* Modules Used:
  - os
  - sys
  - json
  - time
  - signal
  - spotipy
  - threading
  - webbrowser
  - lyricsgenius
